About This Project

This is a simple static website served from a Docker container. You can use Docker to package your web application and deploy it anywhere with ease.

How to Run

  1. Build the Docker image:
    docker build -t static-docker-website .
  2. Run the container:
    docker run -d -p 8080:80 static-docker-website
  3. Open your browser and visit http://localhost:8080
